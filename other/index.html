ClosedGroupFunctions.group_element_calculate — Method
group_element_calculate(expr::String, group::Bijection{String, T}, identity_element::T) -> key::String

Calculate the expression expr using the labels defined in group. The group identity_element is necessary.

Usage:

  1. Let G be a labelled group.
  2. Let I be the identity element from G.
  3. Define a shorthand: Gcalc(expr::String) = group_element_calculate(expr, G, I)
  4. Use Gcalc()`.
ClosedGroupFunctions.default_alphabet — Method
default_alphabet()

Defines a default list of characters that are used to label the generators. Maximum is currently 103 symbols. The default alphabet consists of 'a':'z', 'A':'Z', '𝒜':'𝒵' and 'α':'ω'.
